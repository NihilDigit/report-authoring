# report-formats/ 贡献指南

这里每个文件描述一种报告格式的**内容组织方式**，**不**提供 docx 模板（模板由学校/公司定）。

## 新增一种格式

文件名：`<kind>.md`（小写，短横线分隔）。建议覆盖：

1. **典型结构** — 一般包含哪些章节 / 表格
2. **每章常见写法** — 第一段写什么、图从哪来
3. **默认排版约定** — 字号、行距、样式
4. **图表数量参考** — 一次报告大概多少图多少表
5. **篇幅参考** — 避免写得太短或太啰嗦
6. **典型坑** — 这类报告特有的问题（比如毕设要查重率、课程设计要交答辩 PPT 等）
